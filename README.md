# Mavo Catalog Intelligence Dashboard

This project is a data-driven music analytics dashboard that breaks down how Mavo’s catalog performs across streaming platforms.

## What this shows

Instead of just total streams, this dashboard explains:

- How each song performs over time
- Which songs generate long-term revenue vs short-term spikes
- How audience geography is distributed
- How retention and decay patterns affect catalog value
- Estimated revenue generated per song

## Key Insights

- The catalog is made up of different performance archetypes:
  - Slow-burn retention tracks
  - Fast spike / high decay tracks
  - Balanced growth tracks
  - Stable long-tail catalog assets

- Nigeria remains the dominant market (~40–60%)
- US and diaspora markets drive additional revenue expansion
- The main driver of long-term value is retention, not just virality

## Revenue Model

This dashboard includes a simplified revenue estimation model:

- Spotify: $0.0035 per stream (estimated)
- YouTube: $0.0012 per view (estimated)

This helps translate streaming performance into approximate earnings per track.

## Purpose

This project is designed to help labels, managers, and artists understand:

- What drives long-term catalog revenue
- How songs behave beyond release day
- Where monetization opportunities are strongest
